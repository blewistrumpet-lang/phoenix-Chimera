================================================================================
FORMANT SHIFT CALCULATION
================================================================================

Formula: formantRatio = pow(2.0, (gender - 0.5) * 2.0 * 0.5)

Examples:
  gender = 0.0 (Male):   formantRatio = 0.707  (-6 semitones)
  gender = 0.5 (Neutral): formantRatio = 1.0   (0 semitones)
  gender = 1.0 (Female):  formantRatio = 1.414 (+6 semitones)

================================================================================

GENDER TRANSFORMATION:
  Male → Female:   +6 semitones (one tritone up)
  Female → Male:   -6 semitones (one tritone down)

AGE TRANSFORMATION (at neutral gender):
  Child:     +3 semitones
  Teen:      +1.5 semitones
  Adult:     0 semitones
  Mid Age:   -1 semitone
  Elderly:   -2 semitones

COMBINED (Female + Child):
  Total shift: +9 semitones (major 6th)

================================================================================
TECHNICAL DETAILS
================================================================================

ALGORITHM: Time-domain resampling with sinc interpolation
WINDOW: Hann window for overlap-add
HOP SIZE: 512 samples
FFT SIZE: 2048 samples
OVERLAP: 50%
INTERPOLATION: Windowed sinc (32-tap kernel)

VOLUME COMPENSATION:
  Higher formants: gain *= 1.0 / sqrt(formantRatio)
  Lower formants:  gain *= sqrt(2.0 - formantRatio)

================================================================================
TROUBLESHOOTING
================================================================================

PROBLEM: No effect heard
SOLUTION: Check that Mode = 0.0 (Gender Bender mode)
          Check that Intensity > 0.0
          Check that Gender ≠ 0.5 (neutral position)

PROBLEM: Voice sounds robotic
SOLUTION: Reduce extreme settings
          Use Intensity < 1.0 for blending
          Avoid gender = 0.0 or 1.0 for subtle effects

PROBLEM: Clicks/pops
SOLUTION: Ensure proper prepareToPlay() call
          Check sample rate is correctly set
          Avoid rapid parameter changes

PROBLEM: Too much/too little effect
SOLUTION: Use Intensity parameter to control blend
          Try intermediate Gender values (0.25, 0.75)

================================================================================
LIMITATIONS
================================================================================

• Formant shifting is coupled with pitch shifting
• Not true formant-preserving (future enhancement)
• Quality depends on source material
• Works best with solo vocals
• May introduce artifacts with extreme settings (gender near 0.0 or 1.0)

================================================================================
BEST PRACTICES
================================================================================

1. Start with Intensity = 0.5 for subtle effects
2. Use Age parameter to fine-tune characteristics
3. Avoid extreme Gender values (0.0, 1.0) for natural sound
4. Test with different voice types (bass, tenor, alto, soprano)
5. Use wet/dry blending for more natural results
6. Process at 48kHz or higher for best quality
