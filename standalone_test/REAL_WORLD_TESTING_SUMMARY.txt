TROUBLESHOOTING

Problem: Build fails

Problem: Engine produces silence
Solution: Check processBlock() implementation and parameter initialization

Problem: Excessive clipping
Solution: Review gain staging and add headroom

Problem: Many artifacts
Solution: Check buffer handling, denormals, and memory initialization
